# 성적처리 프로젝트
* 학생의 국어, 영어, 수학 점수를 저장할 model.ScoreVO 클래스 작성
* service.ScoreService 클래스는 작성하여
* List<ScoreVO> 리스트를 선언하고
* makeScore() method 에서 각 과목에 50 ~ 100 까지 임의의 정수를 생성하여 저장하고 List를 20개 작성
* scorePrint() 메서드에서 성적리스트 출력
```
============================================
학번		국어		영어		수학		총점		평균
--------------------------------------------
1 		90		90		90		270		90.0
============================================
		100		00 		00 		00  	00
============================================
```
